A game about hauntings and bureacracy.

Apparition Apparatuses is a Twine game about bureacracy and hauntings. It is a meditation on free will, dead ends, and lost potential.

process

The game is a dead end. It used to be much shorter; it was a thread I followed until my interest turned elsewhere, and I left the story to gather dust for quite some time. It was extended a few years after initial release, after I realized that in order for the game to be a true rumination on rot, it first needed to rot itself.
